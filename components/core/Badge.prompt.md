Inline status/category label. Always pill-shaped. Use for application stages, record states, and categories.

```jsx
<Badge variant="success" dot>Approved</Badge>
<Badge variant="warning" dot>Under Review</Badge>
<Badge variant="destructive">Rejected</Badge>
<Badge variant="primary">Portugal</Badge>
<Badge variant="secondary" size="sm">New</Badge>
<Badge variant="outline">Citizenship by Investment</Badge>
```

Variants map to semantic meaning:
- `success` — approved, active, verified
- `warning` — pending, in review, awaiting documents
- `destructive` — rejected, expired, failed
- `primary` — categorisation, country labels
- `secondary` — highlights, new items
- `default` — neutral, unformatted labels
