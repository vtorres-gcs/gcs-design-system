Buttons

GCS button system — sharp corners (radius 0), uppercase labels, Heebo Medium 14px. Primary uses Night Blue 400; hover transitions to Electric Blue 400.

Variants

Primary · Secondary · Outline · Ghost · Link · Destructive

Sizes

SM 36 · MD 40 · LG 44 · XL 48

States

Default · Hover · Focus · Disabled

With Icons

On Dark Surface

Secondary and Outline are tuned for dark backgrounds

Usage

Primary — main action on a light surface. Hover swaps Night Blue → Electric Blue.

Secondary — main action on dark surfaces; hover inverts to white + Night Blue text.

Outline — tertiary action; default uses Night Blue 50 stroke, hover fills white with Night Blue text.

Ghost / Link — low-emphasis inline actions. Destructive — irreversible actions only.
